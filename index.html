Aufgaben Projektaufgabe 1:

Erstellen Sie die Homepage ihrer Webpräsenz. Folgende Inhalte sollen realisiert werden:

  • Kurzer Text mit mehreren Abschnitten und mehreren Überschriften
  • Eine geordnete verschachtelte Liste
  • Tab Titel

Aufgaben Projektaufgabe 2:

Erstellen Sie mehrere Webseiten, die über eine Navigation verfügen. Folgende Inhalte sollen in mindestens einer Webseite realisiert werden:

  • Bilder einbetten auch mit größen Angaben
  • Ein Link zu einem Video

Aufgaben Projektaufgabe 3:

Stellen Sie für Ihr Webdesign Stilregeln auf. Erstellen Sie eine css Datei und binden diese in allen Webseiten ein. Das Design bleibt Ihnen überlassen.


Aufgaben Projektaufgabe 4:

Nun Layouten Sie Ihre Webpräsenz.

  • Erstellen Sie eine Kopf und Fußzeile
  • Erstellen Sie Spalten und Zeilen mittels Layout Befehle

Aufgaben Projektaufgabe 5:

Nun erweitern Sie Ihre Webpräsenz um eine Tabelle.

  • Fertigen Sie eine Tabelle in html an. Dieses wurde in dem Tutorial nicht angesprochen. Sie müssen es sich selbst aneignen
